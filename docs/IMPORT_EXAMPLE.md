# 卡牌导入示例

## 快速开始

1. 下载模板文件（在导入界面点击"下载模板文件"按钮）
2. 按照模板格式填写卡牌数据
3. 在编辑器中点击"📥 导入卡牌"按钮
4. 选择填写好的Excel文件
5. 查看导入结果

## 字段填写说明

### 卡牌类型
- **锚点卡**：会添加到指定的游戏阶段中
- **随机卡**：会添加到随机事件库中

### 选项影响格式
使用中文维度名+数值，多个影响用逗号分隔：
- `经济+10`：经济维度增加10
- `民生-10`：民生维度减少10
- `经济+10，民生-10`：经济增加10，民生减少10

支持的维度：
- 经济
- 民生
- 生态
- 乡风

### 阶段名称
- 如果填写了阶段名称，锚点卡会添加到对应名称的阶段
- 如果阶段不存在，会自动创建新阶段
- 如果不填写，会添加到第一个阶段

### 关键人物
- 系统会根据人物名称自动创建或匹配NPC
- 如果NPC已存在，会复用现有NPC
- 如果NPC不存在，会自动创建，使用默认头像

## 常见问题

### Q: 如何批量导入多个阶段的卡牌？
A: 在"阶段名称"列中填写不同的阶段名称，系统会自动分组并添加到对应阶段。

### Q: 选项影响可以留空吗？
A: 可以，留空表示该选项不影响游戏数值。

### Q: 如何修改导入后的卡牌？
A: 导入后可以在编辑器中像编辑其他卡牌一样进行修改。

### Q: NPC头像如何设置？
A: 导入时使用默认头像，可以在编辑器的资源库中手动修改NPC的头像和详细信息。
